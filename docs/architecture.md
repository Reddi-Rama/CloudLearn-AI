# CloudLearn System Architecture
This document describes the high-level system components and layout rules for CloudLearn.
## Folder Directory Mapping
- `/frontend`: Next.js 15 app router setup.
- `/backend`: Express-powered API endpoint controller layers.
- `/database`: Database setup containing migrations, schema files, and seeding modules.
- `/docs`: Markdown records detailing deployment steps and design workflows.
- `/scripts`: DevOps building automation files.
